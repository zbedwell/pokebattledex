## Section 3: Feature Breakdown

### Pokemon Directory
`Browse and filter Pokemon by typing, abilities, and core battle stats to find viable options quickly.`

### Pokemon Detail Pages
`Review base stats, matchup summaries, abilities, notable moves, and related alternatives in one tactical profile view.`

### Moves Directory
`Search and filter battle-relevant moves by type, category, power, accuracy, and utility context.`

### Move Detail Pages
`Inspect full move metadata, readable effect text, and Pokemon that can learn the move.`

### Abilities Directory
`Browse abilities with concise summaries and Pokemon counts to identify commonly useful mechanics.`

### Ability Detail Pages
`Understand each ability's practical battle impact and which Pokemon can run it.`

### Type Tool
`Calculate offensive and defensive outcomes for one or two defending types with clear result grouping.`

### Compare Tool
`Compare 2 to 4 Pokemon with side-by-side stats, matchup cues, role tags, and winner highlights.`

### Global Search
`Find Pokemon, moves, abilities, and types from anywhere in the app with grouped suggestions.`
